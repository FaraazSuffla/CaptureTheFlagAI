# Capture The Flag AI

[![Unity](https://img.shields.io/badge/Unity-2022-000000?logo=unity&logoColor=white)](https://unity.com)
![C#](https://img.shields.io/badge/C%23-239120?logo=csharp&logoColor=white)

A Unity game where AI agents compete in a Capture The Flag scenario.

## Features
- AI agents using Unity NavMesh
- Flag capture and defend mechanics
- Two competing teams

## Tech Stack
- Unity 2022
- C#
- Unity NavMesh AI

## Getting Started
1. Clone the repo
2. Open in Unity 2022+
3. Open the main scene and press Play
